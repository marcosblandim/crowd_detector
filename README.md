# crowd_detector
* captura frame de camera local ou remota, a cada n segundos
* faz a detecção de aglomerações nesse frame e o guarda em uma pasta
* variáveis
    - fonte do frame
    - número de frames
    - intervalo entre frames
    - nove do arquivo
    - nome da pasta
